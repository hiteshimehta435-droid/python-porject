# Personal Data Collector

## Project Name
Fundamental Booster - Interactive Personal Data Collector

## Objective
This project collects personal information from the user, processes it, and displays the information using basic Python concepts.

## Features
- Takes user input using input()
- Displays information using print()
- Uses variables and different data types
- Demonstrates type casting
- Uses arithmetic operators
- Calculates birth year from age
- Shows type() and id() functions

## Inputs
- Name
- Age
- Height
- Favourite Number
